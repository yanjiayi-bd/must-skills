# History Courseware Generator

历史学科交互式课件自动生成工具，帮助教师快速创建高质量的历史课件。

## 功能

### 1. 内容生成
- 历史事件梳理
- 人物关系图谱
- 时间轴自动生成
- 地图变化演示

### 2. 交互元素
- 问答互动
- 拖拽排序
- 点击探索
- 动画演示

### 3. 多媒体整合
- 历史图片
- 纪录片片段
- 音频讲解
- 3D 文物模型

### 4. 课件格式
- PPT / Keynote
- PDF 讲义
- 网页交互课件
- 视频课程

## 使用

```bash
# 生成课件
history-courseware "中国古代史 - 秦汉时期" --grade high-school

# 指定格式
history-courseware "工业革命" --format ppt --slides 20

# 生成时间轴
history-courseware "二战进程" --type timeline --interactive

# 人物专题
history-courseware "秦始皇" --type biography --detail full

# 导出多种格式
history-courseware "文艺复兴" --export ppt,pdf,video
```
