# AI Learning Roadmap

为你定制 AI 学习路线图，提供每阶段学习资料、作业项目。从新手，到大师！

## 功能

### 1. 个性化规划
- 评估当前水平
- 确定学习目标
- 制定时间计划
- 推荐学习路径

### 2. 阶段划分
- **新手期** (0-3个月): 基础概念 + 简单实践
- **进阶期** (3-6个月): 深入算法 + 项目实战
- **熟练期** (6-12个月): 专业领域 + 开源贡献
- **专家期** (1年+): 前沿研究 + 创新应用

### 3. 资源配置
- 推荐课程和视频
- 经典论文和书籍
- 实践项目建议
- 社区和资源链接

### 4. 进度追踪
- 学习打卡
- 作业提交
- 能力评估
- 里程碑达成

## 使用

```bash
# 生成个性化路线图
ai-roadmap --goal "成为机器学习工程师" --level beginner --time 6months

# 指定领域
ai-roadmap --field "computer-vision" --current "会Python基础"

# 查看特定阶段
ai-roadmap --phase intermediate --resources

# 更新进度
ai-roadmap --update --completed "基础数学" --next "线性代数"

# 获取推荐项目
ai-roadmap --projects --difficulty medium
```
