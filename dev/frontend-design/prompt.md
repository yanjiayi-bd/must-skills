# Frontend Design

创建独特、生产级的优质前端界面，生成富有创意、精良的代码，避免通用 AI 美学。

## 特点

- 🎨 **独特设计** - 避免千篇一律的 AI 风格
- 💎 **生产级代码** - 直接可用的专业代码
- 📱 **响应式** - 完美适配各种设备
- ♿ **可访问性** - 符合 WCAG 标准
- ⚡ **性能优化** - 轻量高效

## 支持框架

- HTML / CSS / JavaScript
- React / Vue / Angular
- Tailwind CSS
- Styled Components
- SASS / LESS

## 使用

```bash
# 生成完整页面
frontend-design "创建一个现代化的个人博客首页"

# 生成组件
frontend-design --component "导航栏" --style modern
frontend-design --component "卡片列表" --style minimal

# 指定技术栈
frontend-design "仪表盘界面" --tech react --ui tailwind

# 从图片生成
frontend-design --from-image design.png --tech vue

# 优化现有代码
frontend-design --optimize ./components/Header.jsx
```

## 设计系统

```bash
# 生成完整设计系统
frontend-design --design-system "品牌名称"

# 输出包括：
# - 颜色系统
# - 字体规范
# - 组件库
# - 布局网格
# - 动画规范
```

## 避免的问题

- ❌ 过度使用紫色渐变
- ❌ 千篇一律的插画风格
- ❌ 不合理的间距
- ❌ 忽视可访问性
- ❌ 性能糟糕的动画

## 最佳实践

- ✅ 根据品牌定制色彩
- ✅ 有意义的动画
- ✅ 清晰的视觉层次
- ✅ 一致的间距系统
- ✅ 暗色模式支持
